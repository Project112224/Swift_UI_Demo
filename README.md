## SwiftUI Demo

此專案採用 MVVM 架構，頁面只有 Login 與 Home 頁

帳號：1個大寫英文字 + 8位數字
密碼：
- 大小寫
- 數字
- 8碼以上


## 環境

OS: macOS Sonoma 14.5
Xcode: 15.3
